## Frameworks שיווקיים

| Framework | מתי להשתמש |
|-----------|------------|
| **PAS** (Problem → Agitate → Solution) | פוסטים שמדברים לכאב |
| **AIDA** (Attention → Interest → Desire → Action) | פוסטי מכירה |
| **Storytelling** | case study, לפני/אחרי, סיפור אישי |
| **Social Proof** | ביקורות, תוצאות, מספרים |
| **Scarcity** | "נשארו X מקומות" |
| **Authority** | הסמכות, ניסיון, מומחיות |
| **Problem/Solution** | הסבר על שירות/מוצר |

## חוקי תוכן — תמיד

✅ שפה עברית חמה ואנושית
✅ CTA אחד ברור לכל פוסט
✅ Before/After עם תיאור תהליך
✅ מספרים ספציפיים ("3 טיפולים", "6 שבועות")
✅ הדגש: מקצועיות + חמימות + תוצאות אמיתיות

❌ קלישאות: "מהפכה", "ניסים", "מדהים", "חייבת לראות"
❌ הבטחות לא ריאליות
❌ יותר מ-CTA אחד בפוסט
❌ תוכן גנרי ללא קשר לעסק הספציפי
