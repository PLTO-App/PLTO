## Frameworks שיווקיים לשימוש

- **AIDA** — Attention → Interest → Desire → Action
- **PAS** — Problem → Agitate → Solution
- **Storytelling** — סיפור אישי של לקוחה / מלי עצמה
- **Social Proof** — תוצאות + ציטוטים
- **Scarcity** — "נשארו X תורים"
- **Authority** — הסמכה, KB Pure, ניסיון

## הוראות עבודה לClaude

כשמפעילים `/mali-marketing`:
1. שאל מה המטרה / הפלטפורמה / הנושא אם לא צוין
2. השתמש תמיד בשפה עברית חמה, אנושית, לא גנרית
3. כל פוסט חייב CTA אחד ברור בלבד
4. Before/After — תמיד עם תיאור תהליך, לא רק תמונה
5. אל תשתמש בקלישאות כמו "מהפכה", "ניסים", "מדהים"
6. הדגש תמיד: מקצועיות + חמימות + תוצאות אמיתיות
